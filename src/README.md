Projeto Spring Boot para CRUD de tarefas. Data exigida registrada: **12/12/2025** (em `data.sql`).

## Executar
1) Criar banco no MySQL:
   CREATE DATABASE IF NOT EXISTS tarefasdb;
2) Rodar a API:
   mvn spring-boot:run

## Endpoints
- POST /tarefas
- GET /tarefas
- GET /tarefas/{id}
- PUT /tarefas/{id}
- DELETE /tarefas/{id}
